# 🍪 wheres_my_duzzonku (두쫀맵)

두바이 쫀득 쿠키(두쫀쿠)를 시작으로  
두쫀모, 두쫀김밥 등 **두쫀 시리즈를 판매하는 매장을 지도에서 한눈에 찾을 수 있는 위치 기반 웹 서비스**입니다.

SNS 유행으로 생긴 메뉴 특성상  
“지금 어디서 파는지”, “근처에 있는지”를 찾기 어렵다는 문제에서 출발했습니다.

---

## 🧭 프로젝트 개요

**두쫀맵**은  
두쫀 시리즈를 판매하는 매장을 수집하여  
👉 **카카오맵 기반으로 시각화**하는 프로젝트입니다.

- 내 위치 기반 탐색
- 지역명 검색
- 두쫀 시리즈 종류별 필터링

을 핵심 기능으로 하는 **포트폴리오용 MVP 웹 서비스**입니다.

---

## 🎯 개발 목적 (Educational Purpose)

본 프로젝트는 단순 서비스 구현이 아니라  
**웹 프론트엔드 및 외부 API 연동 학습**을 목적으로 제작되었습니다.

### 학습 목표
- React 기반 웹 프로젝트 구조 이해
- Vite 개발 환경 사용 경험
- 외부 지도 API(Kakao Map) 연동
- Web API 인증 및 콘솔 설정 흐름 이해
- 위치 기반 서비스 기본 구현
- Git / GitHub 기반 프로젝트 관리

👉 기존에 주로 데스크탑/서버 개발을 해왔기 때문에,  
**웹 + 지도 + 프론트엔드 스택을 보완하기 위한 학습용 프로젝트**입니다.

## ✨ 주요 기능

### 1️⃣ 지도 기반 매장 조회
- 카카오맵을 이용한 매장 위치 표시
- 매장별 마커 표시

### 2️⃣ 내 위치 기반 탐색
- 브라우저 위치 권한 사용
- 현재 위치 중심 지도 이동

### 3️⃣ 지역명 검색
- 지역명 입력 시 지도 이동
- 해당 지역 매장만 필터링

### 4️⃣ 두쫀 시리즈 필터
- 두쫀쿠 / 두쫀모 / 두쫀김밥
- 복수 선택 가능
- 선택된 메뉴를 판매하는 매장만 표시
